Example name: WebORB Data Management for Flex - Data Deletion
Description: The example demonstrates the usage of the WDMF API to delete a database record. Data deletion in WDMF is very simple. An instance of an active record class represents a row in a database table. Every generated active record class has the remove() method responsible for deletion of the record from the database. The example displays a list of all products in the Product table. To delete a record, select a row in the data grid and click the 'Delete Record' button.
Client-side code: Browse (use Back button to go back)  Download
Server-side code: No custom server-side code is required for this example. All the supporting code is generated by the WDMF code generator.
Things to try:
