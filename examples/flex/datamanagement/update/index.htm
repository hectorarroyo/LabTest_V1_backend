Example name: WebORB Data Management for Flex - Data Update
Description: The example demonstrates the usage of the WDMF API to update data in the database. All WDMF managed objects monitor changes to their internal state. Saving the changes requires one method call: activeRecordInstance.save(). For example, the code below saves the currently selected item in the datagrid:

var selectedProduct:Product = Product( gridProducts.selectedItem );
selectedProduct.save();

Additionally, all active record instances contain the IsDirty flag which can be queried to check if the instance has any pending changes.

The example displays a list of all products from the Product table. The Name and Price columns are editable. A product can be modified either directly in the data grid or using the data-bound text fields. The Save buttons are bound to the IsDirty flag of the corresponding Product instance.
