Example name: WebORB Data Management for Flex - Data Paging
Description: The example demonstrates the the paging feature of the WDMF data retrieval API. All data retrieval functions generated and supported by WDMF support data paging. Configuring a paged data retrieval is as simple as specifying the page size when requesting data. For example, the following call retrieves all product records using the paged approach:

       ActiveRecords.Product.findAll( {PageSize:5} );

The request instructs WDMF to load data five records at a time. The result of the request can be bound to a UI control as the data provider. When the UI control requests additional records, WDMF automatically fetches the corresponding page.

  • The example uses the following WDMF APIs:
    • DataMapper.findBy (custom data retrieval)
    • DataMapper.findBySql
