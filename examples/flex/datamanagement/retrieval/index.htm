Example name: WebORB Data Management for Flex (WDMF) - Data Retrieval
Description: The example demonstrates the usage of various data retrieval operations available in WDMF. Specifically:
  • Retrieval of all records from the database (findAll)
  • Retrieval of a record using raw SQL
  • Retrieval of records using dynamic method
Client-side code: Browse (use Back button to go back)  Download
Server-side code: No custom server-side code is required for this example. All the supporting server-side code is generated by the WDMF code generator.
Things to try:
  • Change the price range in the slider: the example automatically fetches corresponding records through a WDMF query
  • Type in a product name in the text field and click 'Search' - the example uses a dynamic method to find the product
  • Clear the text field and click 'Search' - the example fetches all available products
  • Fetch all available products, then run the 'Data Creation' example in a separate browser window. Create a new product. Notice the product list in this example is automatically updated
